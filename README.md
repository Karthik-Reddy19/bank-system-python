# Python Bank System

Object-Oriented Programming (OOP) was used to create this Python-based banking system project. Basic banking functions like deposits, withdrawals, and balance checks are simulated.

## Features
Make an account
Make a deposit.
Take out cash
Verify the balance of your account.
A straightforward menu-driven interface

## Technologies Employed
Python
Classes and Objects (OOP)

## ⁠️ How to Operate
First. Go to the project folder.
Two. Launch the Python program:
Bank_sys.py in Python
Third. Adhere to the displayed menu options.

## 📌 Description
This project shows how to use Python to implement a basic banking system. To handle account information and carry out tasks like deposits and withdrawals, it makes use of classes.
